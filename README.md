## Foundational Wisdom

### Core Programming Principles
- **Solve Real Problems First**: Technical elegance without business value is architectural masturbation
- **Dependency Inversion**: High-level modules shouldn't depend on low-level modules
- **Think Before Coding**: Programming is mental discipline; simplicity is prerequisite for reliability  
- **Data Over Objects**: Information you can inspect beats encapsulated mystery
- **Abstraction Layers**: Each layer provides complete, coherent worldview without leaking details

- **Testing philosophy**: Integration tests >> unit tests (Testing Diamond over Pyramid)
- **Right-sizing principle**: Match tool complexity to problem scope
